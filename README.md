# Super Store Sales Dashboard

## Overview
This project is an interactive **Power BI Sales Dashboard** created using the Super Store retail dataset.  
The dashboard provides insights into sales performance, profit analysis, customer segments, regional sales, shipping modes, and product categories.

The main objective of this project is to perform business analysis and visualize key performance indicators (KPIs) for better decision-making.

---

## Dashboard Features

- Total Sales Analysis
- Profit Tracking
- Quantity Sold Analysis
- Monthly Sales & Profit Trends
- Sales by Category and Sub-Category
- Regional Sales Distribution
- Sales by Payment Mode
- Sales by Ship Mode
- Interactive Filters/Slicers

---

## Key Insights

- Total Sales: **1.6M**
- Total Quantity Sold: **22K**
- Total Profit: **175K**
- West region generated the highest sales contribution.
- Consumer segment contributed the highest share of sales.
- Standard shipping mode was used most frequently.
- Office Supplies category generated maximum sales.
